Cara mengganti foto background hero di halaman beranda:

1. Siapkan foto dengan ukuran minimal 1920x1080 px (landscape)
2. Simpan foto dengan nama: hero-bg.jpg
3. Letakkan di folder ini: public/images/hero-bg.jpg
4. Refresh halaman — foto langsung tampil sebagai background

Tips:
- Format yang didukung: JPG, PNG, WebP
- Ukuran file disarankan < 500KB agar halaman cepat
- Foto akan otomatis di-overlay dengan warna biru gelap semi-transparan
  sehingga teks tetap terbaca meski foto terang
